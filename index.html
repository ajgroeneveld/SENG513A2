Data Analytics Studies from BTMA 559

BTMA 559 is a class offered by the Haskayne School of Business as an introduction to the R programming language and statistical analysis. This site highlights some of the projects and assignments that were completed during that time.

The tools used are as follows:


Logo Tool Description
R Programming Language
The R programming language is a statistical analysis language used to analyze data
RStudio
RStudio is the top IDE for R development
Selenium
Selenium is a web automation application, that when used with R can enable scraping of Javascript populated tables
Google Maps API
The Google Maps API when used with R can load maps and overlay data points to enable data visualization over maps
